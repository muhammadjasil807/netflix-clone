Netflix Clone (HTML & CSS)
A responsive **Netflix landing page and sign-in page UI clone** built using **HTML5 and CSS**.
This project focuses on layout design, responsiveness, and clean UI structuring

Project Overview:
This project recreates the **Netflix homepage** and **Sign In page** to practice modern web design concepts such as:
* Clean HTML structure
* Reusable CSS components
* Overlay effects
* Flexbox layouts
* Responsive design using media queries

This is a **front-end UI clone only**. No real authentication or backend functionality is included.

Technologies Used:
HTML5 – Page structure
CSS – Styling and layout
Flexbox – Alignment and layout control
Media Queries – Mobile responsiveness

Pages Included:
Home Page (index.html)
* Netflix-style hero section
* Dark overlay background
* Navigation bar with logo, language selector, and sign-in button
* Email input with call-to-action

Sign In Page (signin.html)
* Centered login card
* Email and password fields
* Sign-in options
* Remember me checkbox
* Navigation back to sign-up page

UI Features:
* Full screen background image
* Dark transparent overlay for readability
* Netflix inspired red color theme
* Reusable button and input styles
* Clean typography hierarchy

Responsive Design: css
The layout adapts to different screen sizes using CSS media queries:
* Text scales properly on mobile
* Form layout stacks vertically on small screens
* Login box adjusts width for tablets and phones


what do we learn
* Understanding real-world UI layouts
* Practicing responsive web design
* Improving CSS structuring skills
* Multi page website linking
